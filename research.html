Abstract
Abstract

The exponential growth of digital data stored on personal computers often leads to storage shortages and system performance degradation. Non-technical users face significant challenges in identifying space-consuming files, removing duplicate content, and cleaning temporary system data. Consequently, effective storage management has become a critical usability issue for the average consumer.

This paper introduces DiskVision, a desktop utility for the Windows operating system designed to simplify disk storage management. Leveraging artificial intelligence via integration with the Ollama framework, DiskVision provides intelligent recommendations by classifying files into three safety categories: Safe, Caution, and Dangerous.

Beyond intelligent cleanup, DiskVision offers high-speed file search, duplicate detection via hashing algorithms, file compression, and AES-256 file encryption. The software monitors real-time CPU and RAM usage and manages startup programs through a bilingual (Arabic/English) interface.

Section I
Introduction

In recent years, personal computers have become repositories for vast amounts of digital data. As users download files and install programs, storage space is progressively consumed by unnecessary data. Temporary files, duplicate content, outdated downloads, and unused applications frequently accumulate on systems without user awareness. Over time, this data accumulation can significantly degrade computer performance.

Users commonly encounter low disk space notifications, system slowdowns, and difficulty locating important files. Many applications also launch automatically during startup, consuming valuable memory resources. While modern operating systems include basic disk management utilities, these tools fail to offer clear insights into actual storage utilization patterns.

The concept of disk management software has a well-established history. Early solutions, such as Norton Utilities (1982), assisted users in DOS environments. Microsoft introduced Disk Cleanup with Windows 98 (1998). The early 2000s brought popular tools like CCleaner (2003) and WinDirStat (2003).

Building upon these, DiskVision integrates modern technologies with intelligent data analysis through several specialized modules: a file scanning engine, duplicate detection using hashing algorithms, disk space visualization, and a real-time system monitor. A key innovation is its integration of artificial intelligence to deliver intelligent recommendations, minimizing the risk of accidentally deleting critical data.

Section II
Related Work

Disk management software constitutes a critical category of utility applications. As modern computers accumulate vast data, storage devices inevitably become congested. The following sections examine the most significant existing tools and compare them with DiskVision.

A. CCleaner

CCleaner [1] enhances performance by removing unnecessary files and cleaning temporary data. It provides startup program management and a Windows Registry cleaning tool. Despite its popularity, CCleaner primarily focuses on system cleaning and lacks advanced disk visualization, comprehensive duplicate detection, or AI-driven recommendations. DiskVision encompasses all of CCleaner's capabilities while extending with disk usage analysis, hash-based duplicate detection, real-time monitoring, and intelligent AI-powered file recommendations.

B. WinDirStat

WinDirStat [2] visualizes disk usage with a Treemap visualization, where colored rectangles correspond to relative file sizes. It provides a hierarchical folder view for file system navigation. However, WinDirStat's functionality is essentially limited to disk space visualization — no system cleaning, duplicate detection, startup management, or real-time monitoring. DiskVision integrates WinDirStat's treemap concept within a comprehensive suite, allowing users to take immediate action from the same unified interface.

C. Duplicate Cleaner

Duplicate Cleaner [3] employs the MD5 hashing algorithm to generate unique content-based fingerprints, identifying identical files even with different names across disparate folders. While effective for duplicate detection, its functionality remains confined to this single task — no disk visualization, system cleaning, or performance monitoring. DiskVision integrates the same MD5-based detection within a broader platform combining disk analysis, cleanup, monitoring, and AI recommendations.

D. TreeSize

TreeSize [4] facilitates disk space analysis by scanning directories and presenting results in a hierarchical tree structure with sorting by size, type, or date. Like WinDirStat, it functions as a passive analysis tool — no cleaning tools, duplicate detection, startup management, or CPU/memory monitoring. DiskVision provides equivalent analysis while incorporating all active management features plus AI-based recommendations.

E. Everything Search

Everything Search [5] delivers rapid file search by maintaining a persistent index of the NTFS Master File Table, yielding near-instantaneous results. However, Everything is purely a passive search tool — no analysis, cleanup, duplicate detection, or AI functionality. DiskVision incorporates a comparable SQLite-backed indexing engine while combining search with active management features.

F. Summary and Comparison

Existing tools each concentrate on limited subsets: CCleaner cleans but doesn't detect duplicates; WinDirStat and TreeSize visualize but don't clean; Duplicate Cleaner detects duplicates only; Everything searches but takes no action. None incorporate AI-based analysis. DiskVision comprehensively addresses all these limitations within a single unified application.

Section III
Methodology

This section describes DiskVision's system design, main components, technologies, and step-by-step workflow. The system is designed as a complete disk management solution combining multiple tools within one application.

A. System Overview

DiskVision is built using Python 3 and runs on Windows 10/11. The Tkinter library provides the GUI, making it accessible without technical knowledge. The interface supports Arabic and English with dark and light themes. The system is organized into independent modules sharing data through the main application controller.

C. System Components
  • File Scanner Module: Recursive disk scan via pathlib, collecting path, size, and file type.
  • Duplicate Detector Module: Two-phase detection — size grouping then MD5 hashing.
  • Disk Space Analyzer: Hierarchical treemap of folder sizes for visual analysis.
  • Temp Cleaner Module: Scans Windows Temp, browser caches, and prefetch folders.
  • AI Recommendation Engine: Connects to Ollama (LLaMA 3.2) for file classifications.
  • Resource Monitor: Real-time CPU, RAM, and process memory via psutil.
  • Startup Manager: Queries Windows Registry Run keys and the Startup folder.
  • File Encryption Module: AES-256-CTR + HMAC authentication.
  • File Compression Module: ZIP/LZMA with optional 7-Zip integration.
  • Graphical User Interface: CCleaner-inspired layout with sidebar navigation.
D. Technologies Used
  • Python 3 — main programming language.
  • Tkinter — GUI library for all windows and panels.
  • psutil — real-time CPU and RAM data.
  • hashlib — MD5 hashing for duplicate detection.
  • pathlib — file system navigation and scanning.
  • SQLite — local database for file search index.
  • Ollama REST API — connection to locally-running LLM.
  • zipfile / pyaes — compression and AES encryption.
  • send2trash — safe deletion to Recycle Bin.
  • winreg — Windows Registry startup entries.
E. AI Analysis Modes
  • Classification Mode: Labels each file SAFE, CAUTION, or DANGEROUS with a reason.
  • Summary Mode: Disk usage report with top three storage-freeing recommendations.
  • Cleanup Plan Mode: Four-step plan — immediate deletion, review, keep, future prevention.
  • Smart Delete Mode: Returns only paths of definitely-safe files (tmp, log, cache).
Section IV
Conclusion

This paper presented DiskVision, a desktop application for Windows providing intelligent disk analysis and storage optimization. The system addresses the gradual accumulation of unnecessary files that reduces storage space and degrades system performance — a common problem for everyday computer users.

DiskVision combines hard disk scanning, MD5-based duplicate detection, treemap visualization, temporary file cleaning, fast SQLite-indexed search, file encryption, file compression, real-time CPU/RAM monitoring, and startup program management within a single application.

The most important distinguishing feature is the integration of artificial intelligence. Through the Ollama framework and locally-running LLMs, DiskVision classifies files as SAFE, CAUTION, or DANGEROUS, generates disk usage summaries, creates structured cleanup plans, and identifies smart deletion candidates — helping users make better decisions without accidentally deleting critical files.

The comparison with CCleaner, WinDirStat, Duplicate Cleaner, TreeSize, and Everything Search demonstrated that DiskVision is the only solution combining all capabilities in a single platform with a bilingual Arabic/English interface.

Future improvements include cloud storage support (Google Drive, OneDrive), scheduled automatic cleaning, perceptual hashing for near-duplicate image detection, and cross-platform support for macOS and Linux. DiskVision demonstrates that integrating AI into system utility software is both practical and beneficial for everyday users.
